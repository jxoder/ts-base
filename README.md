## Typescript boilerplate

### Prerequisites

- direnv
- Node.js 22.13.0 (ref .nvmrc)
- pnpm
- typescript 5.7.3 (ref package.json)

### Install

```bash
pnpm install
```

### Run

```bash
# run watch mode
pnpm start:dev
```

### Test

config in `test/jest.config.json`

```bash
# Run all tests
pnpm test
```
